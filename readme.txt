1. Description:

This project aims to develop a medical assistant chatbot that asks patients about their symptoms and related information, and generates a PDF report for the senior doctor. The chatbot also saves the patient's data in the database.

2. Installing libraries/modules

pip install fpdf
pip install SpeechRecognition
pip install pyttsx3
pip install edge-tts
pip install deep-translator
pip install ipython
pip install pygame
pip install pymongo

3. Commands to run the project

(i) Install the necessary libraries in VS Code.
(ii) Download the program and open it in VS Code.
(iii) Change the path on line 57,435 according to the location of symotoms_1.json .

(iv) Adjust the file path on line 77 to specify where you want to save the generated audio outputs, ideally in a designated folder.

(v) Again adjust the file path on line 243 to specify where you want to save the PDF report generated.

(vi) Now you can run the project.
